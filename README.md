# Aviles Storage - Storage made simple

The Storage Module is a JavaScript module that provides an interface for managing client-side storage. It simplifies the process of storing and retrieving data in local storage, session storage, cache storage, indexDB storage, and cookie storage.

## Features

Manage data using basic CRUD operations in:

1. Local Storage
2. Session Storage
3. ~~Cache~~ - Coming Soon
4. ~~IndexDB~~ - Coming Soon
5. ~~Cookies~~ - Coming Soon

## Getting Started

### Installation

You can include the Storage Module in your project by downloading the storage.js file and adding it to your project directory. Then, include the following script tag in your HTML file:

```html
    <script src="path/to/storage.js"></script>
```

### Usage

Each different type of storage has it's own object:

* Local Storage - lcl
* Session Storage - sess
